# Wubba Lubba dub dub 🛸

Aplicación web para explorar y guardar personajes de Rick y Morty.

## 🚀 Inicio rápido

### Requisitos
- Node.js 16+
- npm o yarn

### Instalación

```bash
# Clonar repo
git clone https://github.com/osflores21/wubba-lubba-dub-dub.git
cd wubba-lubba-dub-dub

# Instalar dependencias
npm install

# Iniciar servidor
npm run dev
```

## ✨ Características

- 🔍 **Búsqueda** de personajes por nombre
- 🏷️ **Filtros** por estado (Alive, Dead, Unknown)
- ⭐ **Favoritos** - guardar tus personajes preferidos
- 📄 **Paginación** - navega entre 42 páginas
- 🎯 **Modal** con detalles completos del personaje
- 📱 **Responsive** - funciona en todos los dispositivos

## 🛠️ Tecnologías

- React 18 + TypeScript
- Mantine UI
- Axios
- Phosphor Icons

## 📖 Cómo usar

1. **Buscar** - escribe el nombre del personaje
2. **Filtrar** - selecciona el estado (Alive, Dead, Unknown)
3. **Navegar** - usa los botones de paginación
4. **Ver detalles** - haz clic en cualquier card
5. **Favoritos** - haz clic en la estrella para guardar


## 📁 Estructura

